## 二、 核心架构：双层 AI 判定系统
我们将逻辑与情绪彻底解耦，确保 AI 不剧透：

> **[玩家问题]** -> **[Judge 层]** (持有汤底) -> 输出：`是 | 否 | 接近 | 无关`
>
> **[Host 层]** (包装回复) -> 拿到结果 -> 套用“臭宝”语料 -> **[最终回复]**

### ⚡ 性能优化决策：
* **「否」与「无关」**：直接从预设 `meme_pool` 秒回，不调用 AI。
* **「是」与「接近」**：调用 Host AI 进行情绪渲染，增加关键时刻的“等待感”。

## 三、 物证收藏与社交货币 (The Viral Hook)
* **发放逻辑**：每关结案必掉落一张物证卡。
* **稀有度算法**：由破案步数决定（≤10步：**Legendary 金色**）。
* **存档找回**：支持 **Base64 序列码** 导出/导入（无需数据库，10行代码搞定同步）。

## 四、 四大工程“红线”决策 (The Big 4)
1.  **安全红线**：API Key 必须走 **Next.js API Route** 转发，前端严禁直接调用。
2.  **合规红线**：Host 层加 **关键词拦截器**，防止 AI 秃噜嘴泄露关键信息。
3.  **留存红线**：**连续 5 次「无关」** 自动触发“渣男式”提示，防止玩家产生挫败感。
4.  **传播红线**：Case_001 物证设定为 **“长裤套短裤微缩模型”**，用抽象驱动分享。
